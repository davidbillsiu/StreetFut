StreetFut

Um jogo 2D de futebol estilo street desenvolvido com HTML, CSS e JavaScript, focado em mecânicas simples, movimentação fluida e interação dinâmica entre jogador e bola.

Demonstração

(adicione aqui prints ou gif do jogo rodando)

Como executar o projeto
Clone o repositório:
git clone https://github.com/davidbillsiu/StreetFut.git
Acesse a pasta:
cd StreetFut
Abra o arquivo:
index.html

Ou simplesmente dê dois cliques no index.html.

Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla JS)
Funcionalidades
Controle de jogador
Sistema de movimentação da bola
Colisão entre jogador e bola
Sistema de vidas
Mecânica estilo jogo 2D arcade
Cenário estilo campo de areia / street
